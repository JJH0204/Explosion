## 문제 설명
플래그 생성 시스템
사용자가 직접 플래그를 생성하고 관리할 수 있는 시스템입니다. 

생성된 플래그는 데이터베이스에 안전하게 저장되며, 이를 통해 사용자는 자신만의 플래그를 만들 수 있습니다.

## 힌트
1. userDB에 접근하여 자신의 테이블 구조를 확인하세요.
2. 플래그가 저장되는 필드를 확인하세요.
3. SQL 터미널을 통해 플래그가 정상적으로 저장되었는지 확인할 수 있습니다.
4. 플래그 생성 후에는 반드시 저장 여부를 확인하세요.

## 학습 목표
- 데이터베이스 테이블 구조 이해
- SQL 쿼리를 통한 데이터 조회 및 수정
- 웹 기반 데이터베이스 관리 시스템 활용
