## 문제 설명
404 에러 페이지처럼 보이는 웹 페이지에 숨겨진 인증 시스템이 있습니다.  

페이지의 소스 코드를 분석하고 Base64 인코딩된 올바른 인증 키를 찾아내세요.

## 목표
- 난독화된 JavaScript 코드 분석
- Base64 인코딩/디코딩의 이해
- 올바른 인증 키를 찾아 플래그 획득

## 제한사항
- 별도의 도구나 프로그램 없이 브라우저만으로 해결 가능
- 브루트포스 공격은 필요하지 않음

## 힌트
1. 페이지의 소스 코드를 자세히 살펴보세요.
2. 개발자 도구의 콘솔 메시지를 확인하세요.
3. JavaScript의 atob() 함수에 대해 알아보세요.
