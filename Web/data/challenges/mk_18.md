## 문제 설명
네트워크 192.168.1.0/24을 동일한 크기의 4개 서브넷으로 나누는 문제입니다.
각 서브넷의 네트워크 주소, 브로드캐스트 주소, 사용 가능한 IP 범위를 구하고 이를 입력합니다.

## 입력 조건
1. 네트워크 주소와 브로드캐스트 주소를 정확히 계산하세요.
2. 각 서브넷의 사용 가능한 첫 번째 및 마지막 IP 범위를 구하세요.
3. 서브넷 마스크를 정확히 설정하세요.
4. 모든 입력은 다음 형식에 맞게 작성해야 합니다:
네트워크-브로드캐스트: 192.168.1.x-192.168.1.x
호스트 범위: 192.168.1.x-192.168.1.x

## 출력 조건
모든 서브넷 설정을 올바르게 계산하고 입력하여 최종 플래그를 획득하세요.

## 힌트
- 네트워크 주소 및 브로드캐스트 주소는 호스트 수 요구 사항을 기반으로 계산해야 합니다.
- 서브넷 마스크는 호스트 수에 따라 변경됩니다.
- 라우터 인터페이스 IP는 각 서브넷의 유효 IP 범위 안에서 선택해야 합니다.
