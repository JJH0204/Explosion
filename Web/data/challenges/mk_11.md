## 문제 설명
매 3분마다 새로운 당첨 번호가 생성됩니다.  
1부터 45까지의 숫자 중에서 6개를 선택하여 당첨 번호를 맞춰보세요.  
운이 좋다면 플래그를 획득할 수 있을지도 모릅니다!  

## 게임 규칙
1. 1부터 45까지의 숫자 중 6개를 선택합니다.
2. 매 3분마다 자동으로 당첨 번호가 추첨됩니다.
3. 선택한 6개의 번호가 당첨 번호와 정확히 일치하면 플래그가 주어집니다.

## 기능
- 수동 선택: 원하는 번호를 직접 클릭하여 선택
- 자동 선택: 랜덤으로 6개의 번호를 자동 선택
- 초기화: 선택한 번호를 모두 지우고 다시 시작
- 추첨 기록: 이전 추첨 결과 확인 가능

## 힌트
1. 매 3분마다 새로운 기회가 주어집니다.
2. 당첨 확률은 1/8,145,060 입니다... 정말일까요?
3. 대다수의 프로그래머는 의사난수를 생성할 때 시간을 활용합니다.
