## 문제 설명
FileHub라는 파일 업로드 서비스에서 발견된 보안 취약점을 이용하여 관리자 권한을 획득하는 문제입니다. 

## 입력 조건
- Base64로 인코딩된 플래그 복호화 필요

## 출력 조건
관리자 패널에 접근하여 암호화된 플래그를 복호화하세요.

## 힌트
1. 파일 업로드 시 허용되는 확장자 목록을 확인하세요
2. 업로드한 파일의 내용이 어떻게 검사되는지 분석하세요
3. Base64 디코딩 과정을 이해하세요

## 플래그 형식
플래그는 Base64로 인코딩되어 있으며, 복호화하면 얻을 수 있습니다.
