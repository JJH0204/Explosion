## 문제 설명
2D 점프킹 게임을 통해 다양한 플랫폼을 탐험하며 단서를 찾고, 최종 목표에 도달하는 문제입니다.

플레이어는 페페 캐릭터를 조작하여 플랫폼을 점프하며 올라가야 합니다. 

각 플랫폼에는 단서가 숨겨져 있으며, 이를 통해 최종 목표를 달성해야 합니다.

## 학습 목표
- 2D 게임 물리 엔진의 기본 이해
- 점프 및 충돌 감지 알고리즘 학습
- 게임 내 단서 탐색 및 문제 해결 능력 개발

## 단계별 과제
1단계: 시작 지점에서 첫 번째 단서 찾기
2단계: 중간 플랫폼에서 추가 단서 수집
3단계: 최종 목표에 도달하여 문제 해결

## 필요한 도구
- 웹 브라우저 (게임 실행)
- 기본적인 JavaScript 이해 (게임 코드 분석)
