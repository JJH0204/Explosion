## 문제 설명
SQL Injection 취약점이 있는 로그인 페이지입니다.  

사용자 입력값이 SQL 쿼리에 직접 삽입되는 취약점을 이용해 관리자 권한을 획득하세요.  

## 입력 조건
- SQL Injection 구문 분석  
- 데이터베이스 구조 파악  
- 관리자 계정 정보 획득  

## 출력 조건 
SQL Injection 취약점을 이용해 관리자로 로그인하여 플래그를 획득하세요.  

## 힌트
1. SQL 구문 오류를 발생시켜 취약점을 확인하세요  
2. 테이블 구조를 조회하는 구문을 사용해보세요  
3. 로그인 우회 구문을 시도해보세요  
4. 관리자 계정 정보를 찾아보세요
