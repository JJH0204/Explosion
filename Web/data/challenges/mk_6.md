## 문제 설명
반응속도를 테스트하는 웹 게임입니다.

화면이 초록색으로 바뀌는 순간을 정확하게 포착하여 빠르게 클릭하세요.

100ms 미만의 반응속도를 달성하면 플래그를 획득할 수 있습니다.

## 입력 조건
- 빨간색 화면을 클릭하여 게임 시작
- 초록색으로 바뀔 때까지 대기
- 초록색으로 바뀌면 최대한 빨리 클릭
- 너무 일찍 클릭하면 실격

## 출력 조건 
100ms 미만의 반응속도를 달성하여 플래그를 획득하세요.

## 힌트
1. 화면 전환에 집중하세요.
2. 마우스 클릭 위치를 미리 고정하세요.
3. 여러 번 시도하여 최고 기록을 경신하세요.
4. 모니터 주사율이 높을수록 유리합니다.
