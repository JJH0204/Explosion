## 문제 설명
일반 사용자용 로그인 페이지가 있습니다. 

프록시 도구나 개발자 도구를 사용하여 일반 사용자 권한을 관리자 권한으로 상승시켜 플래그를 획득하세요.

## 목표
- 프록시 도구를 사용하여 HTTP 요청 분석
- 권한 관련 파라미터 조작
- 관리자 권한으로 로그인하여 플래그 획득

## 제한사항
- 브루트포스 공격은 필요하지 않음
- SQL Injection은 필요하지 않음
- 프록시 도구(예: Burp Suite) 또는 브라우저 개발자 도구만으로 해결 가능

## 힌트
1. 로그인 요청을 프록시 툴로 확인해 보세요.
2. role 파라미터의 값을 확인해 보세요.
3. 개발자 도구(F12)의 Elements 탭도 확인해보세요.
