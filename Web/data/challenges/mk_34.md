## 문제 설명
이 문제는 두 개의 악성코드 샘플(Lab05-01.dll과 Lab03-01.exe)을 분석하여 특정 정보를 찾아내는 리버싱 과제입니다. 

각 샘플의 특징과 동작을 파악하여 요구되는 정보를 정확히 찾아내야 합니다.

## 문제 조건
### 1. Lab05-01.dll 분석
- DLLMain의 주소를 찾아내세요.
- 0x10001757 주소에서 발생하는 DNS 요청 주소를 확인하세요.

### 2. Lab03-01.exe 분석
- 호스트 기반 표시자(파일명 또는 레지스트리 키)를 찾아내세요.
- 네트워크 기반 표시자(통신하는 도메인)를 확인하세요.

## 입력 조건
- DLLMain 주소는 16진수 형태로 입력 (예: 0x10001000)
- DNS 요청 주소와 도메인은 정확한 철자로 입력
- 호스트 기반 표시자는 하나만 입력해도 정답입니다.

## 출력 조건
모든 문항에 대해 정확한 답을 입력하면 플래그가 출력됩니다.

## 힌트
1. flare를 사용하여 정적 분석을 수행하세요.
2. Process Monitor로 호스트 기반 활동을 모니터링하세요.
3. Wireshark로 네트워크 트래픽을 분석하세요.
